# 🌴 PearlPara Travels

A full-stack tourism web application for Sri Lanka built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This project delivers an interactive platform for users to explore tour packages, view detailed itineraries, and visualize travel routes, while providing administrators with a secure dashboard to manage content.

## ✨ Features

### 🌍 User Features
- 🧭 Browse tour packages with detailed descriptions  
- 📅 Day-by-day travel itineraries  
- 🗺 Interactive route maps (Leaflet + OSRM API)  
- 🖼 Image gallery with masonry layout & lightbox  
- 📱 Fully responsive design  

### 🔐 Admin Features
- Secure admin login with JWT authentication  
- Manage tours (Add / Edit / Delete)  
- Manage testimonials  
- View and manage contact messages
